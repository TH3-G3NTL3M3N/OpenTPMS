Key Firmware Features

ANT+ Tire Pressure Profile

Broadcasts on the standard ANT+ Bicycle Tire Pressure device profile. Garmin, Wahoo, and other ANT+ head units see it natively. Includes pressure, temperature, and sensor location (front/rear).

BLE Custom GATT Service

Custom service with characteristics for: pressure (notify), temperature (notify), battery level, sensor location, alert threshold, TX interval. Read/write for configuration via Web Bluetooth.

Temperature-Compensated Alerts

Uses ideal gas law (P1/T1 = P2/T2) to calculate expected pressure at current temperature based on baseline reading. Only triggers flat alert when pressure drops beyond thermal explanation. Eliminates false alarms from weather changes.

OTA DFU (Over-The-Air Update)

Nordic's secure DFU bootloader. Firmware updates via nRF Connect app (phone). Signed images prevent unauthorized firmware. User pulls tire, connects via BLE, pushes update.

Non-Volatile Config Storage

Stores in nRF52 flash (FDS): sensor location (F/R), alert threshold PSI, TX interval, baseline pressure/temp for compensation, pairing keys. Persists across battery swaps.

Power Budget

Deep sleep: ~3µA (MCU + accel)
Active TX cycle (3s): ~8mA for ~3ms = 24µAs per cycle
Avg current (1hr ride): ~15µA
Est. battery life: 2-3 years (1hr/day riding)
