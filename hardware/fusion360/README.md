# Fusion 360 Project Files

The PCB schematic and layout are designed in Fusion 360 Electronics.
The enclosure is designed in Fusion 360 Mechanical workspace.

Fusion 360 project files are cloud-based. Export formats:
- Gerber files: `hardware/gerbers/`
- STEP files: `hardware/enclosure/` (after enclosure design)
- BOM: `docs/OpenTPMS_BOM.xlsx`

Schematic netlist and component placement are verified — see
`docs/fusion360-schematic-guide.md` for the full design walkthrough.
